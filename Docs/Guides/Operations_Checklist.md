# Operations Checklist

## Pre-Run

- Battery charged and connector tight
- RF modules seated and antenna orientation checked
- Tank line and nozzle line leak-check completed
- Wheels rotate freely and no cable snags
- Pump MOSFET heat path verified

## Start Sequence

1. Power transmitter.
2. Power main unit.
3. Confirm indicator LEDs and serial logs.
4. Set speed and flow level.
5. Engage clutch and test one direction at low speed first.

## During Run

- Keep clear line-of-sight where possible
- Start at low flow and ramp gradually
- Avoid deep mud until traction is validated
- Observe low-battery and water-level indicators

## Shutdown

1. Send stop command.
2. Turn off pump output.
3. Power off main unit.
4. Power off transmitter.
5. Flush spray line and nozzles with clean water.

## Post-Run Maintenance

- Dry electronics compartment
- Inspect motor terminals and driver board
- Clean and store nozzles
- Recharge batteries to storage-safe level when idle
